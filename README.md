# Particle Playground

A real-time interactive particle playground built with C++ and SFML, featuring multiple particle simulations, customizable visual effects, and interactive controls.  

Explore different particle behaviors ranging from magical motion to fireworks, all rendered in real time.

---

## ✨ Features
### Particle Modes
- Magical
- Freefall
- Constrained
- Wave
- Freeze
- Fireworks

### 🔨Customization
- Adjustable particle spawn rate
- Firework color palette
- Toggleable controls panel
- About screen
- Gravity toggle (mode-specific)
- Random particle sizes
- Dynamic color palettes
- Twinkling particle effect

### 💻Performance
- Real-time FPS counter
- Live particle counter
- Efficient particle lifetime management

## 🎮 Controls

| Key | Action |
|-----|--------|
| Left Click | Emit particles |
| 1 | Magical Mode |
| 2 | Free Fall Mode |
| 3 | Constrained Mode |
| 4 | Wave Mode |
| 5 | Freeze Mode |
| 6 | Fireworks Mode |
| G | Toggle Gravity |
| C | Clear Particles |
| Up / Down | Increase / Decrease Spawn Rate |
| H | Toggle Controls Panel |
| Tab | Toggle Color Palette (mode-specific) |

---

## 🛠Built with
- C++
- SFML 2.5.1
- Visual Studio

## 🎯 Purpose

This project began as a simple particle generator and gradually evolved into an interactive particle playground for experimenting with different particle behaviors, rendering techniques, and real-time visual effects.
